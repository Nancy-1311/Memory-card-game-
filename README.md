# Memory Card Game

This is a simple memory card matching game made with HTML, CSS, and JavaScript.  
The goal is to flip cards and find all the matching pairs.

## Features
- Cards flip when clicked  
- Cards are shuffled every time the game starts  
- Matching cards stay open  
- Unmatched cards flip back after a short delay  
- Restart button to play again  
- Works on mobile and desktop  

## How to Play
1. Click a card to flip it.  
2. Flip another card and try to match the images.  
3. If both cards match, they stay face up.  
4. If they don’t match, they flip back.  
5. Keep going until all pairs are matched.  
6. Press **Restart** to play again with a new shuffle.

## Files
- **index.html** – main structure  
- **style.css** – all styling and layout  
- **index.js** – game logic and functions  

## Run the Game
Open the `index.html` file in your browser.
