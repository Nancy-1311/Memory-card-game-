# Memory Card Game This is a simple memory card matching game made with HTML, CSS, and JavaScript. The goal is to flip cards and find all the matching pairs. ## Features - Cards flip when clicked - Cards are shuffled every time the game starts - Matching cards stay open - Unmatched cards flip back after a short time - Restart button to play again - Works on both mobile and desktop screens ## How to Play 1. Click a card to flip it. 2. Flip another card and try to match the images. 3. If both cards match, they stay face up. 4. If they don’t match, they flip back. 5. Continue until all pairs are matched. 6. Press “Restart” to shuffle and play again. ## Files - **index.html** – main page - **style.css** – design and layout - **index.js** – game logic ## Run the Game Just open index.html in your browser. ## GitHub Pages (Live Link) https://github.com/Nancy-1311/Memory-card-game-
